Featured Project

AI Collaborative Coding Platform

The Problem

Developer environments require synchronous editing capability without merging conflicts. Traditional tools lack built-in semantic context to assist developers locally with docstring generation, syntax checks, and autocomplete recommendations in real-time.

The Approach

Constructed a secure full stack workspace utilizing WebSockets to sync cursor locations and document diffs immediately. Integrated LLM inference pipelines that receive file buffers, run prompt-tuned context assembly, and return completions asynchronously to prevent editing lags.

Tech Stack

Node.js Express Socket.io Python backend API Gateway

Key Metrics & Impact

  • Asynchronous latency clocked under 150ms for suggestions.
  • Implemented 3 smart modules: Auto-Docstrings, Syntax Checker, and Code Explainer.
  • Handled 50+ concurrent active socket channels with automatic cleanup mechanisms.

Featured Project

AI Resume Relevance Check System

The Problem

Hiring pipelines are flooded with resumes that are hard to filter manually. A mechanism was required to quickly cross-examine resumes against detailed job descriptions to rank applicant suitability programmatically.

The Approach

I developed a text-processing engine in Python using natural language processing (NLP). The model parses PDF text, runs TF-IDF vectorization and cosine similarity calculations, and scores matching attributes while highlighting missing core keyphrases.

Tech Stack

Python Scikit-learn NLTK PyPDF2 HTML5 UI

Key Metrics & Impact

  • 94% matching accuracy measured against test recruitment benchmarks.
  • Reduced parsing and sorting workflows to under 1 second per document.
  • Generated visual relevance matrices displaying keyphrase overlaps.
